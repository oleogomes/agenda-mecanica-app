Mecânica Turbo Bayer
Quem somos

Nós somos a Mecânica Turbo Bayer, uma mecânica fictícia, criada exclusivamente para a disciplina de projeto tecnológico do curso de Tecnologia em Sistemas para a Internet da ULBRA - Universidade Luterana do Brasil, câmpus Gravataí/RS. Nosso idealizador é o aluno Leonardo Gomes.

As tecnologias utilizadas para desenvolver essa aplicação foram:
- Frontend: Angular 10 / Html / Css / TypeScript
- Backend: Java 8 / Springboot 2.1.8 / JsonWebToken
- Banco de dados: PostgreeSql

Agenda

O objetivo desta agenda é facilitar o agendamento de serviços em mecânica. De um lado está o cliente, que não irá precisar ligar, mandar mensagem no Whatsapp ou qualquer outro tipo de contato para agendar um serviço, nem para saber valores e quanto tempo demora um determinado serviço. De outro lado está a mecânica, que através dos mecânicos, consegue visualizar a agenda diária e se preparar para atender melhor os clientes, e através do administrador, consegue visualizar a base de clientes, sua equipe com a quantidade de serviços realizada por cada membro, visualizar todos os serviços agendados e já realizados, adicionar novos mecânicos a equipe, visualizar e adicionar novos tipos de serviço para agendamento.
